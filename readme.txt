Adds the all new Twitter.com button to your website.

== Description ==

This plugin adds the all new Twitter.com button to your website.

* Choose whether to use RT or via
* Choose position of button
* Choose style of button

**The easiest to install Official Twitter.com button!**

== Installation ==

1. Upload `new-twitter-button.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Options can be found under settings.
